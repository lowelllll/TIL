# AWS(Amazon WEB Service)
> 클라우드 서비스 플랫폼으로서 컴퓨팅 파워, 데이터베이스 스토리지, 콘텐츠 전송 및 기타 기능을 제공함.

### 클라우딩 컴퓨터
- 구름(인터넷)위에 설치되어있는 컴퓨터.
- 인터넷에 연결되어있는 거대한 컴퓨터를 사용함.

### 장점
- 선투자금 없음.
- 규모의 경제
- 탄력적인 처리용량
- 속도와 민첩성
- 비지니스에만 집중  
    <i>개발 팀은 수 많은 서버를  관리하느라 시간을 허비하지않고 자신들의 고객들을 위해 차별화된 가치를 추구하는데 집중할 수 있음.</i>
- 글로벌 진출
