# Struct 
>구조화된 데이터를 처리할 때 사용함. 구조화되었다는 말은 의미가 연결되어 한 덩어리로 처리하는 방식을 말함.  

### 배열의 단점 
비슷한 속성을 가진 것들끼리 묶을 수 있지만 타입이 모두 같아야한다는 단점이 있다.

### Struct 특징
- 비슷한 속성을 가진 데이터를 구조화하여 사용할 수 있다.  
- 클래스 개념과 비슷하지만 메소드를 가진 클래스와 달리 속성만을 가지고 있음.

### 선언
```c
struct Man {
  char name[50];
  int  age;
  char gender;
  char tel[50];
};
```
인간의 정보를 구조화한다면 위처럼 묶어 구조화 할 수 있다.

### 구조체 사용
```c
struct Man man; // 구조체 변수
man.age = 18; // 변수에 값 할당.
```
