# 전자 계산기 만들기

전자 계산기를 만들기 위해서는 평소 사용하는 중위 표기식 대신 후위 표기식을 사용해 계산함.

## 후위 표기법

#### 중위 표기법

> 연산자가 피연산자들 사이에 위치 ex) 1+5+2

#### 후위 표기법

> 연산자가 피연산자들 뒤에 위치 ex) 15+2+



중위 표기식 `A + B * C` 는 후위 표기식 `A B C * +`



### 후위  표기법 알고리즘 설계 

1. 반복문으로 문자열 하나씩 조건을 비교함
2. 피연산자이면 그냥 출력
3. `(`여는 괄호이면 스택에 push
4. `)` 닫는 괄호이면 스택에서 `(`이 나올때 까지 pop하고 피연산자들은 출력.
5. 연산자이면 스택의 마지막 연산자가  현재 연산자보다 우선순위가 높거나 같을  때  pop하고 출력. 현재 연산자는 push
6. 수식이 끝났으면 스택에 남아있는 연산자를 pop한 후 출력.



### 후위 표기 수식 계산 알고리즘 설계

1. 피연산자면 스택에 push
2. 연산자 만나면 pop을 2번하고 해당 연산자에 맞게 계산 후 결과 값을 스택에 push.
3. 수식이 끝나면 스택에 남은 마지막 값이 최종 결과 값!
