# vim 명령어 

### 입력

i   커서 있는데서 입력 

a  커서 있는줄 끝에서 입력 

o 커서 있는줄 아래 빈줄 삽입 

O 커서 있는줄 위에 빈줄 삽입 



### 이동

H 화면 맨 위로 이동 

M 화면 중간으로이동 

L 화면 맨 아래로 이동 

$ 줄 끝으로 이동 

0 줄 처음으로 이동 



### 삭제

x 한문자 삭제 

dd 한줄 삭제 

dw 한 단어 삭제 

u 바로 전에 수행한 명령을 취소 



### 문자열 찾기

/[word] 단어 찾기 

n 다음 name으로이동 

N 역방향으로 이동 



### 문자열 대체

:s/str/rep 현재 행의 str을 rep으로 대체 

:%s/str/rep/g 파일 전체 str을 rep으로 대체 



### 기타

:set nu 행번호 보여주기 

:set nonu 행번호 보여주지않기
